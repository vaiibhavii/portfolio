# Public Folder

Place static assets here that need to be publicly accessible.

## Required Files

### resume.pdf
- Add your resume PDF here
- Name it exactly `resume.pdf`
- This will be downloaded when users click "Download Resume"

### Optional Files

You can also add:
- Favicon files (favicon.ico, apple-touch-icon.png, etc.)
- og-image.png (for social media previews - recommended size: 1200x630px)
- robots.txt
- Any other static assets

## Usage

Files in this folder are served from the root path `/`.

Example:
- `/public/resume.pdf` → accessible at `https://yoursite.com/resume.pdf`
- `/public/favicon.ico` → accessible at `https://yoursite.com/favicon.ico`
